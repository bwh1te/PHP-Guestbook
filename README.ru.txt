СОДЕРЖИМОЕ АРХИВА
=================
1. Файлы приложения:
    classes/
        auth.php - класс авторизации, работает с БД через общий объектный интерфейс приложения
        dbadapter.php - адаптер для создания подключения к БД по параметрам, заданным в общем виде
        guestbook.php - потомок modeldriver, расширенный под конкретную задачу
        modeldriver.php - объектный интерфейс для работы с БД
        page.php - конструирует веб-страницу из шаблона
    
    config/
        dbconfig.txt - параметры подключения к базе данных
        err_settings.txt - настройки вывода ошибок php
        settings.txt - настройки приложения
        
    controllers/
    Обработчики запрашиваемых страниц.
        ban.php - блокировка пользователя
        delete.php - удаление поста
        login.php - авторизация
        logout.php - выход
        newpost.php - сохранение нового поста
        not_found.php - 404 // не отлажена
        showpost.php - вывод постов, основная функция
        
    models/
    Модели объектов, хранящихся в БД.
        model.php - супер-класс для моделей
        post.php - модель поста
        user.php - модель пользователя
        
    templates/
    Шаблоны веб-страниц
        css/
        Таблицы стилей
            main.css - основная таблица стилей
        img/
        Изображения
            logo.png - логотип
        showpost.html - шаблон вывода постов
        
    tests/
    Тесты. Содержит только один тест для примера.
        test_dbadapter.php
    
    utils/
    Вспомогательные функции
        autoloader.php - автозагрузчик классов контроллеров
        configreader.php - читает файлы конфигурации и сохраняет настройки в массив
        
2. Файлы инсталлятора:
    install.php - установочный файл, запускать только после прописывания настроек
    test_data.txt - тестовые данные, на их основе будут созданые первые посты
        pagetoshow.php - основываясь на текущей странице, возращает список страниц, на которые надо дать ссылки

УСТАНОВКА
=========
1. Скопировать содержимое папки app в /var/www
2. Создать базу данных (в примере - guestbook)
3. Прописать настройки БД в файле config/dbconfig.txt
4. Запустить install.php
